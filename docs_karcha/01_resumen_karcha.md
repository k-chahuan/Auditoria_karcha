# Resumen Ejecutivo - Auditoría de Seguridad Web
**Organización:** Universidad del Pacífico
**Portal Auditado:** Portal de Clientes (Alumnos, Docentes y Postulantes)
**Nivel de Alcance:** Entorno de Pruebas Controlado (DVWA - Nivel Low)
**Auditor:** Departamento de Informática

## Contexto de Negocio
La Universidad del Pacífico es una institución de educación superior que gestiona una alta densidad de datos sensibles, incluyendo registros académicos (notas, asistencias), información financiera (créditos, becas, datos de pago) y datos personales de identificación (RUT, direcciones, antecedentes socioeconómicos). El "Portal de Clientes" actúa como la interfaz crítica para la toma de ramos, visualización de estados de cuenta y carga de evaluaciones. Un compromiso en este sistema no solo afecta la confidencialidad, sino que puede alterar la integridad académica y paralizar la continuidad operativa institucional.
